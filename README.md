【導入】
　Assets以下に"コンパイル済み"フォルダ内の"Pluginフォルダ"を入れるだけで使えます。
 
 ※任意の階層のPluginフォルダにUnityUtilityLib.dllをいれてください


【ライブラリの内容】
・SoundManager及びSoundList
・FadeManager及び、そのEditor
・拡張メソッドが書かれているクラス(関数、コルーチン、Action)
※Timeクラスの拡張は使用用途が限られるため廃止しました。

【使い方】
名前空間は"Yuuki"で区切っています。
usingするなり、含めるなりして使ってください。

詳細は配布した資料を参照してください。
